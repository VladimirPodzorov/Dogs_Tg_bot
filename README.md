# Python_basic_diploma
## Описание. Телеграм бот: "Собачий бот".
Телеграм бот для тех, кто хочет найти собаку по определенным критериям.<br/>
В данном боте поиск идет по размеру животного: вес и рост.
А также по дополнительным параметрам, таким как: на сколько сильно собака линяет, на сколько является шумной, 
на сколько сложно дрессировать и на сколько способна собака к охране.

Поиск собак идет, путем обращения к открытой API сайта<br/> https://api-ninjas.com/api/dogs

### Запуск программы
Проект разработан на языке Python, и запускается с помощью клонирования репозитория и установки необходимых
библиотек (pip install -r requirements.txt).<br/>

Для запуска необходимо получить *"api ключ"* на сайте https://api-ninjas.com/api <br/>
и токен телеграм-бота. Как получить токен можно узнать по ссылке https://core.telegram.org/api  <br/>
После получения ключей, скопировать их и вставить в файл *.env* в корневой папке проекта.
```
TOKEN_TG = 'телеграм токен'
API_key = 'api ключ сайта'
```
Бот запускается из файла ***main.py***, находящегося в корневой папке проекта.

### Коммуникация пользователя с ботом
Пользователь может коммуницировать с ботом через команды в меню интерфейса или из чата бота.<br/>
Список команд, которые пользователь может использовать:

1. /start - запуск бота
2. /help - описание команд
3. /history - вывод истории поиска (последние десять запросов)
4. /low - вывод списка собак по минимальным значениям: веса и роста, с возможностью выбора дополнительных параметров (см. описание)
5. /high - вывод списка собак по максимальным значениям: веса и роста, с возможностью выбора дополнительных параметров (см. описание)
6. /custom - вывод показателей пользовательского диапазона по значениям веса и роста собаки


#### Команды /low, /high и /custom позволяют получить информацию:
* Минамальный вес
* Максимальный вес
* Минимальный рост
* Максимальный рост
* Продолжительность жизни (от и до)
* Изображение собаки

А также на сколько по пятибальной шкале собака:
* линяет
* шумит (голосит)
* обучается (способность к дрессировке)
* способность собаки к охране
* дружелюбность с детьми
* дружелюбность с другими собаками
* энергичность породы

### Папки проекта
**database** - содержит модель для работы с базой данных.
Взаимодействует с командой /history, для получения истории поиска.

**siteAPI** - содержит файлы с функциями по которым идет запрос к сайту Api для поиска:
1. ***dog_finder.py*** для запросов по командам /low и /high
2. ***cust_req.py*** для запроса по команде /custom

**tgAPI** - содержит клавиатуры интерфейса, логику вывода сообщений в чат, код работы телеграм-бота
